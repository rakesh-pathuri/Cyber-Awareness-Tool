# Cyber Awareness Lab

An interactive, responsive educational platform for cyber awareness, featuring both a Student Dashboard for learning and a Teacher Dashboard for presentations.

## Features

- **Interactive Simulations**: Engaging simulators for Phishing, Passwords, Digital Footprint, Deepfakes, and more.
- **Teacher Dashboard**: A robust presentation mode for classroom environments.
- **Student Dashboard**: A self-paced learning portal with interactive modules and quizzes.
- **Responsive Design**: Mobile-friendly layout ensuring a great experience on any device.
- **PWA Support**: Installable as a Progressive Web App (PWA) with offline capabilities.
- **Internationalization (i18n)**: Support for multiple languages.

## Tech Stack

- **Framework**: React + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **PWA**: vite-plugin-pwa

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
